📌 Course Overview
This repository contains the coursework and practical implementations for the Programming Fundamentals using C++ course at the University of Karachi (DCS/UBIT).

The course focuses on structured/procedural programming, providing the foundational logic required for future AI and Machine Learning applications. It uniquely integrates OpenGL/GLUT for data visualization alongside standard console-based applications.

🛠 Tech Stack
Language: C++ (Procedural)

Compiler: g++ (MSYS2 / UCRT64)

Editor: VS Code

Graphics: GLUT/OpenGL

📂 Syllabus Phases
Phase 1: Foundation (Environment setup, I/O, Arithmetic)

Phase 2: Core Constructs (Loops, Decision making, 1D Arrays)

Phase 3: Modularity & Data (Functions, File I/O, 2D Arrays/Matrices)

Phase 4: Algorithms (Linear Search, Greedy Algorithms, Recurrence)

🚀 Key Topics Covered
Mathematical Foundations: Physics equations (Newton's Laws), Trigonometry (Sine waves, Parametric circles).

Visualization: Plotting equations and shapes using glVertex2f().

Algorithmic Thinking: Dry running code, flowchart-to-code conversion, and logic building.
